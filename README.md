# Data Pipeline System

## What It Does

This repository contains modular Python pipelines and exercises for ingesting,
transforming, and structuring datasets for analysis.

## Tech Stack

- Python
- CSV-based data processing
- Data engineering pipeline patterns

## How To Run

1. Create and activate a virtual environment (recommended).
2. Run pipeline scripts directly, for example:
   - `python ex01/ex01.py`
   - `python ex02/ex02.py`
   - `python ex03/ex03.py`
3. For the final project, run:
   - `python Final_project_Oliver_Chandler/main.py`
